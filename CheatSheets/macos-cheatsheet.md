## Mac Screenshot Keyboard Shortcuts
There are 3 ways you can capture a screenshot on macOS using keyboard shortcuts.

| Command                      | Description                   |
| ---------------------------- | ----------------------------- |
| Command (‚åò) + Shift (‚áß) + 3 | Takes a screenshot of the entire screen |
| Command (‚åò) + Shift (‚áß) + 4 | Takes the screenshot of a selected area of the screen or a cropped screenshot|
| Command (‚åò) + Shift (‚áß) + 4 + Spacebar + Click on the window | Takes a screenshot of a specific window.
| Command (‚åò) + Shift (‚áß) + 5 | Take a screenshot or record screen (Mojave onwards) |
| Command (‚åò) + Shift (‚áß) + 6 | Take a screenshot of the Touchbar |
| Command (‚åò) + Shift (‚áß) + Control (‚åÉ) + 3 | Capture the screen to the Clipboard |
